# Overview
Introduction to Algorithms Textbook Implementations in the Zig programming language

## Common Commands

```bash
# run all tests
zig build test

# run specific test suites
zig build test-lib
zig build test-main
zig build test-sorting
zig build test-insertion
zig build test-integration

# generate docs
zig build docs
```
